# Rendix

Rendix is an immediate mode renderer backed by pixi.js@^7.
Inspired by the LOVE2d API.

## Features

- Foundational
  - [x] render
  - [ ] clear
- DrawObjects
  - [ ] drawCircle
  - [ ] drawLine
  - [ ] drawPolygon
  - [ ] drawRect
  - [x] drawSprte
  - [ ] drawText
- Framebuffer
  - [ ] Save Framebuffer
- Matrix Transforms
  - [x] scale
  - [x] rotate
  - [x] translate
  - [x] push
  - [x] pop
